dat export

  Get tabular (parsed) data from a dataset on stdout.

Parameters:

  --dataset=<name> (-d) (required)

    The name of the dataset.

Options:

  --limit=<number>

    The number of rows to output. Defaults to infinity.

  --format=[ndjson, csv, json] (-f)

    How to return the output. Default: ndjson

  --full

    Output full metadata with each row, including autogenerated keys.
